# homebrew-tap

Shared Homebrew tap for krahd command-line tools.

## Install

```bash
brew tap krahd/tap
brew install s2a
```

## Available formulae

- `s2a`: Squarespace to Astro migration CLI
- `mail_summariser`: Local-first mail workflow backend
- `video_glitcher`: Realtime desktop app for glitching video and exporting corrupted MP4 files
